# E-Commerce Analytics Portfolio

End-to-end analytics portfolio demonstrating **MySQL, SQL, Python (Pandas), Power BI, and Tableau Public** skills using an e-commerce dataset.

## What this project demonstrates
- Relational database design in **MySQL**
- SQL querying with **joins, aggregations, CTEs, and window functions**
- Pandas-based validation and feature engineering
- Power BI dashboard development
- Tableau Public portfolio publishing
- Business storytelling with KPI dashboards

## Project Architecture
**MySQL Database → SQL Curated Tables → Python (Pandas) Validation → Power BI / Tableau Dashboards**

## Portfolio Deliverables
### Dashboard 1 — Executive KPI Overview
- Monthly revenue trends
- Orders and Average Order Value (AOV)
- Refund and bad order rate
- Category/product performance

### Dashboard 2 — Customer & Retention Studio
- Customer lifetime value
- RFM segmentation
- Cohort retention
- Product growth insights
