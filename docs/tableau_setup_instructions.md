# Tableau Public Setup Instructions

1. Download Tableau Public from: https://public.tableau.com/en-us/s/download
2. Install and sign in
3. Import CSVs from `data/processed/`
4. Build dashboards:
   - Executive KPI Overview
   - RFM Segmentation
   - Cohort Retention
   - Product Performance
5. Publish dashboards and update the main README with links
